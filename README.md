# IotHomeSecurityApp

This app is build on window 10

Clone repository

Install mongodb
Run MongoDb server

Install node.js and nodemon
Install angular-cli

goto app directory
Run npm install
Run nodemon 

Then open a new commmand line
Goto angular-src in app directory 
run npm install
then 
ng serve

Open browser type localhost:4200 and press Enter

Now app will start
